# Robycam · тахеометр

Веб-калькулятор для супервайзера Spidercam/Robycam: ввод координат с тахеометра, пересчёт по формулам Excel и схема установки.

## Открыть

**https://insoman4ik.github.io/tacheometer-app/app.html**

Страница входа: **https://insoman4ik.github.io/tacheometer-app/**

## Как работает

1. **Выбор точки** — шкивы 1–4, оптика 5 или голова Robycam.
2. **Координаты** — X, Y, H с тахеометра для выбранной точки.
3. **Подтвердить** — координаты записываются, пересчитываются углы Excel и обновляется схема.

## Возможности

- Блоки Excel: линия ∥ оси Y (left/right) и ∥ оси X (nearest/farest)
- Жёлтые ячейки — координаты из подтверждённого ввода; зелёные — ° ′ ″
- 6 точек: шкивы 1–4, оптика 5, голова 6
- Схема в трёх проекциях и длины тросов
- PWA: офлайн после первого открытия

## Формулы

- Линия ∥ Y: `α = arctg((Xправ − Xлев) / (Yправ − Yлев))`, при α &lt; 0 → +360°
- Линия ∥ X: `α = arctg((Yдальн − Yближ) / (Xдальн − Xближ))`, при α &lt; 0 → +360°
- DMS: усечение как `INT()` в Excel
